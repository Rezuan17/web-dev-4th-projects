Some Questions

  • Question: What is DOM?

  • Ans: DOM stands for Document Object Model. It represents the structure of a web page.

  • Question: How to manipulate DOM?

  • Ans: You can manipulate DOM using JavaScript with methods like `getElementById`, `appendChild`, etc.

  • Question: What is the role of DOM in JavaScript?

  • Ans: DOM allows JavaScript to dynamically modify HTML and CSS.
